# Phase 04 UI Design Contract

## Layouts
1. **Monitor 1 (Queue)**: Table format. Columns: STATE, ID, REQ, ESCOLH, FALT, JOB. Substitutions indicated by `(S)` or `(R)` in yellow.
2. **Monitor 2 (Status)**: Three panels (Colony, Operation, Critical Stock). Alerts shown as banners.
3. **Terminal (Editor)**: TUI with options [1] Edit Equivalences, [2] Edit Tiers, [3] Overrides, [4] Allowlist, [0] Exit.
