# Plan 07-01: Auto-geração de config e robustez MP

## Contexto

Ver [07-CONTEXT.md](./07-CONTEXT.md).

## Entregáveis

- `config.ini` ausente → criar automaticamente com defaults.
- Log explícito dos defaults aplicados/criados.
- Checklist de compatibilidade multiplayer e ajustes de diagnóstico.

## Plano

1. Adicionar função para escrever um `config.ini` padrão quando não existir.
2. No bootstrap, detectar ausência e criar o arquivo antes de carregar config.
3. Logar defaults aplicados (chave → valor) quando o arquivo foi gerado.
4. Adicionar testes para o fluxo “sem config.ini”.
